# 🌩️ KYZEN Mutation Tracker

Real-time Roblox Mutation & Weather Tracker.

## ✨ Features

- 📡 Real-time Weather Detection
- 🌧️ Rain
- ⚡ Lightning
- 🌈 Rainbow
- ❄️ Snowfall
- ⭐ Starfall
- 🌌 Aurora
- ☀️ Sunburst
- 🌕 Bloodmoon
- 🪙 Goldmoon
- 🌈🌕 Rainbow Moon
- 💥🌕 Mega Moon

## 📢 Discord Webhook

Automatically sends a notification when a new weather event starts.

Information included:
- Event Name
- Detection Time (HH:MM:SS)

Duplicate notifications are automatically prevented.
